## 思考問題
1. 輸入一個 init，並且輸入 calls 的話，會依 calls 運算
2. 如果是 `.increment()` ← init + 1
3. `.reset()` ← init → init
4. `.decrement()` ← init -1

# Solve
```js
var createCounter = function(init) {
  let n = init;
  return {
    //error
    increment: () => n++,
    increment: () => ++n,
    reset: () => n = init,
    decrement: () => --n,
  }
}
```
