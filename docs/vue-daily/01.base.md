## SFC (Single-File Components 單文件組件)
Vue 的 SFC 有點像 codepen 的組成，將 HTML、CSS、JavaScript 寫在一頁裡面，
另外官方文件有提到常見的『關注點分離』的議題，有興趣可以去官方文件參閱。

1. 檔名：要有一~兩個英文單字組成例 `App.vue` 或 `HelloWorld.vue`
2. 副檔名： `.vue`
3. 基本格式
  - 這裡要記得加上 `setup` 和 `scoped`

## 基本架構
```html
<script setup>
</script>

<template>
	<h1>標題</h1>
</template>

<style scoped>
</style>
```
