## 自動引入？
我們可以將多個 type 型別設定統一整理在一個 `.ts` 中，並且使用 `export` & `import` 的方式來引入到需要的地方；\
但如果有上百個 type 要 `export` 跟 `import` 就會變的很麻煩，\
此時可以在 `tsconfig.ts` 將 type 設定為全域 

## 怎麼做？

```ts title="tsconfig.ts"
//...
// },
"include": ["src/**/*"]
```

這樣就可以直接不需要 `import` 就直接引入 type

```ts title="main.ts"
const user: TUserBase = {
  name: "Clove",
  age: 20,
  userAddress: "Taiwan"
}
```
